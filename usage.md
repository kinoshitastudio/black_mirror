## Fill + LINE（塗りと線）

オブジェクトを選択した状態、または描画前にツールバーの Fill ボタンから設定できます。

### FILL（塗り）

| アイコン | 効果 |
|---------|------|
| `○` | 塗りなし（アウトラインのみ） |
| `●` | 黒塗り |
| `◑` | グレー塗り（半透明） |
| `□` | 白塗り |

### LINE（線）

| アイコン | 効果 |
|---------|------|
| `□` | アウトラインあり（デフォルト） |
| `□` ~~（赤斜線）~~ | アウトラインなし |

> 塗りと線の組み合わせで、ソリッド・アウトラインのみ・フラット・インビジブルなど多様な表現が可能。

## Pen — パスの描き方とオートクローズ

1. `P` キーまたはツールバーの Pen ボタンを選択
2. キャンバスでドラッグして自由パスを描く
3. パスの始点近く（約 22px 以内）まで戻ると **青いスナップリング** が表示される
4. その状態でリリースすると、パスが自動的に閉じる（`closed = true`）
5. 閉じたパスは Fill 設定で塗り色を適用できる

> タッチ操作時はスナップ判定が約 32px に拡大。Fat Finger 対応。

---

## 画像のアップロード

1. ツールバーの `🖼` ボタンを押す（または `I` キー）
2. ファイル選択ダイアログから PNG / JPG / GIF を選択
3. 現在のビューの中央に自動配置（最大辺 480px にリサイズ）
4. Select ツールで移動・リサイズ・回転が可能

画像はすべて base64 で JSON に埋め込まれるため、ネットワーク不要で保存・復元が完結します。

---

## Export PNG（透過書き出し）

1. Select ツールで書き出したいオブジェクトを選択（複数可）
2. `X` キー、またはツールバーの Export ボタンを押す
3. 選択オブジェクトを Retina 解像度（×2）でタイトクロップ、透過背景で PNG ダウンロード

> 何も選択していない場合は全オブジェクトを書き出す。

## テーマモード (v1.0)

Black Mirror Board には **3 つのテーマ** があります。ターミナルにコマンドを打ち込むことで瞬時に切り替わります（グリッチフラッシュ + スキャントランジション付き）。

| コマンド | モード | 背景 | 線／塗り |
|---------|--------|------|---------|
| `light` | Surface  | `#FAFAF8` | `#0c0c0c` |
| `dark`  | Void     | `#000000` | `#F0EDE6` |
| `gray`  | Studio   | `#DCDBD5` | `#1600A2` (cobalt) |

既存のオブジェクトの塗り色は、モード切替時に **自動的に再マップ** されます（`resolveFill()`）。
ライトの黒塗りは Dark で白、Gray でコバルトに再描画。データは常に light-mode の正規値で保持され、描画時に変換されます。

## リサイズ — Shift・Alt モディファイア (v1.0)

Select ツールで選択したオブジェクトの **bbox ハンドル（8点）** をドラッグ中、キーボードの修飾キーで挙動が変化します。

| 修飾 | 効果 |
|------|------|
| なし | 自由リサイズ |
| **Shift** | アスペクト比を完全固定（縦横比保持） |
| **Alt (Option)** | 中心点を固定したまま全方位に拡大縮小 |
| **Shift + Alt** | 中心固定 + アスペクト比固定（複合） |

### 画像のデフォルト挙動

> 画像（🖼）は **コーナーハンドルでデフォルトがアスペクト比固定**。
> Shift を押している間だけ自由リサイズになります。
> 写真の比率が意図せず崩れる事故を防ぐための仕様です。

### スナップ（v1.0）

Shift ドラッグ中は、倍率が **0.5× / 0.75× / 1× / 1.25× / 1.5× / 2× / 3×** のキリの良い値に ±2.5% の窓で吸着します。

## 選択の当たり判定 (v1.0 — Hit-Test 2.0)

細い鉛筆線やパスでも、正確に線の上をクリックする必要はありません。

- すべての線に **見た目の線 + 8px の見えないパディング** が付与されています。
- 鉛筆ストロークは **セグメント距離ベース** で判定されるため、簡略化後の頂点間にあるギャップもヒットします。
- 閉じた塗り付きパスは **内部クリック** でも掴めます。

### ホバーフィードバック

- マウスカーソルが掴めるオブジェクトの判定領域に入ると、**点線のバウンディングボックス** が表示されます。
- カーソルが `move` に変化し、「ここを掴める」ことを明示。
- ハンドルに乗ると `nwse-resize` / `nesw-resize` / `ns-resize` / `ew-resize` が方向に応じて表示されます。

## パス編集（Path Editing）

Select ツールでオブジェクトを選択すると、**青い小さな正方形（アンカーポイント）** が表示されます。

アンカーポイントをドラッグして図形を変形できます。

| 図形 | アンカーの種類 |
|------|--------------|
| ○ Circle | 上・右・下・左の 4 点（半径をドラッグで変更） |
| △ Triangle | 3 つの頂点（任意の位置へ移動可） |
| □ Square | 4 コーナー＋4 辺中点（計 8 点でリサイズ） |
| → Arrow | 始点・終点（方向・長さを変更） |
| ✎ Stroke | 約 10 点のコントロールポイント |
| 🖼 Image | 4 コーナー＋4 辺中点（計 8 点） |

## モバイル操作ガイド

Black Mirror Board は **モバイルでも完全動作** します。

### タップ操作

- **1 本指タップ** → オブジェクトの選択
- **1 本指ドラッグ** → 図形を描く、またはオブジェクトを移動
- **選択後にアンカーをタップ＆ドラッグ** → 形状を変形（タッチ時は当たり判定が約 2.2 倍）

### パネルリサイズ

モバイル画面ではキャンバスとターミナルが縦に分割されています。  
中央の **ドラッグハンドル（⠿）** を上下にドラッグして比率を自由に変更できます。

## プロジェクトの保存・復元

### 保存（SAVE ↓）

ヘッダーの **SAVE ↓** ボタンで現在の状態を `.json` ファイルとしてダウンロード。

**保存される内容:**
- キャンバス上の全オブジェクト（座標・パス・スタイル・塗り・線・回転）
- 画像データ（base64 埋め込み）
- ビューの状態（ズーム・位置）
- ターミナルの出力ログ

### 読み込み（OPEN ↑）

ヘッダーの **OPEN ↑** ボタンで保存済みの `.json` ファイルを選択し、作業を完全復元。

> **ローカル完結型設計。ネットワーク通信は一切行いません。**  
> API キー・認証・アカウント登録すべて不要。

## ターミナルコマンド

| 入力 | 動作 |
